üîπ Step 1: Install MySQL Server
sudo dnf install mysql-server -y
sudo systemctl start mysqld
sudo systemctl enable mysqld

üîπ Step 2: Set password for LOCAL root user

Root exists only as 'root'@'localhost', so use this:

sudo mysql -e "ALTER USER 'root'@'localhost' IDENTIFIED BY 'database@1';"

üîπ Step 3: Login to MySQL
mysql -u root -p


Enter password:

database@1

üîπ Step 4: Create database & table

Inside MySQL:

CREATE DATABASE IF NOT EXISTS school_db;
USE school_db;

CREATE TABLE IF NOT EXISTS student_basic(
  id INT AUTO_INCREMENT PRIMARY KEY,
  school_name VARCHAR(100),
  school_branch VARCHAR(100),
  student_name VARCHAR(100),
  student_roll VARCHAR(50)
);

‚ùó IMPORTANT CHANGE

Do NOT use this (WRONG):

GRANT ALL PRIVILEGES ON school_db.* TO 'root'@'%' IDENTIFIED BY 'database@1';


This is what caused your error.

IDENTIFIED BY inside GRANT is not allowed in MySQL 8.

üîπ Step 5: Create a NEW remote user for remote access

Instead of modifying root (best practice), create an admin:

CREATE USER 'admin'@'%' IDENTIFIED BY 'database@1';
GRANT ALL PRIVILEGES ON school_db.* TO 'admin'@'%';
FLUSH PRIVILEGES;


Now remote access will work reliably.

üîπ Step 6: Enable remote connections in MySQL config
Edit /etc/my.cnf:
sudo vim /etc/my.cnf


Add at the bottom:

bind-address=0.0.0.0

Also check /etc/my.cnf.d/mysql-server.cnf (if exists):
sudo vim /etc/my.cnf.d/mysql-server.cnf


If present, add:

bind-address=0.0.0.0

üîπ Step 7: Restart MySQL
sudo systemctl restart mysqld

üîπ Step 8: Open the port in EC2 Security Group

Go to AWS console ‚Üí EC2 ‚Üí Security Groups ‚Üí Inbound Rules
Add:

Type	Port	Source
MySQL/Aurora	3306	0.0.0.0/0 (or your IP only)
üîπ Step 9: Test remote connection

From your Python script / MySQL Workbench use:

import mysql.connector

# Ask user for the DB host/IP
db_host = input("Enter the host/IP address of the studentapp-db MySQL server: ")

# Database connection config
db_config = {
    'host': db_host,
    'user': 'admin',
    'password': 'database@1',
    'database': 'school_db',
    'port': 3306
}

# Collect basic details from user
school_name = input("Enter School Name: ")
school_branch = input("Enter School Branch: ")
student_name = input("Enter Student Name: ")
student_roll = input("Enter Student Roll Number: ")
#student_class = input("Enter Student Class: ")
#maths_unit1_marks = input("Enter Student maths_unit1_marks: ")

# Insert collected data into DB
conn = mysql.connector.connect(**db_config)
cur = conn.cursor()
cur.execute(
    "INSERT INTO student_basic (school_name, school_branch, student_name, student_roll) VALUES (%s, %s, %s, %s)",
    (school_name, school_branch, student_name, student_roll)
)
conn.commit()
cur.close()
conn.close()

print("Details saved successfully!")
